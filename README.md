# Тестування вебпорталу Державної служби статистики України

**Автор:** Лебедовська Вероніка  
**Опис:** Цей проєкт присвячений функціональному та UI-тестуванню розділу «Публікації» офіційного вебпорталу Державної служби статистики України ([stat.gov.ua](https://stat.gov.ua)). Метою проєкту було забезпечення якості роботи ключових елементів навігації, системи фільтрації даних та валідація користувацького інтерфейсу в різних браузерах.

**Ментальна мапа сайту:** [Mind Map Link](https://mm.tt/map/3996054379?t=XSGOpX1hiD)

---

## 🎯 Мета проєкту
* Перевірка коректності роботи фільтрів «Типи публікацій».
* Оцінка стабільності відображення інтерфейсу (Cross-browser testing).
* Виявлення критичних UI/UX дефектів, що впливають на користувацький досвід.

## 🛠 Інструментарій
Для реалізації завдань використовувалися наступні інструменти:
* **Баг-трекінг:** Jira
* **Тестова документація:** Microsoft Excel (Test Cases, Checklists, Test Summary Report)
* **Планування:** Microsoft Word (Test Plan)
* **Браузери:** Google Chrome, Opera, Microsoft Edge

## 📊 Основні результати
* **Загальна кількість тест-кейсів:** 11
* **Загальна кількість пунктів чек-листа:** 42
* **Pass Rate:** 91% (на основі загальних метрик тест-кейсів)
* **Баг:** Виявлено дефект `[Cookie] Перекриття меню «Типи публікацій» модальним вікном Cookie Banner`, який блокує основний функціонал сторінки.

## 📂 Структура проєкту
Репозиторій містить наступні артефакти:
* `Test Plan.docx` — стратегія та план тестування.
* `Test Cases.xlsx` — набір тест-кейсів.
* `Checklist.xlsx` — перевірочні списки за браузерами.
* `Test Summary Report.xlsx` — підсумковий звіт із метриками.
* `README.md` — опис проєкту та результатів.
